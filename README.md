# Employees

The main frameworks and libraries used to build this project are:

* React
* React-thunk
* Redux Framework
* axios
* .Net Core
* Odata

#Backend

created with .net core and Odata
  - Url Odata: https://localhost:44372/odata/employees
  
#FrontEnd

  Install dependencies with npm
  ```
  npm install
  ```
  
  And run with npm
  ```
  npm start
  ```
